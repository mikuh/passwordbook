# passwordbook
一个去中心化的密码管理工具

为了管理大量的互联网平台账号和密码，开发这个Dapp来管理，只要知道自己的以太坊私钥，所有的密码都在自己的掌控之中。



## 使用方式：

1.下载本项目

2.在根目录，执行 `npm run dev`

3.根据提示地址，用浏览器访问

使用需要安装metamask插件。

填写平台、账号、密码后点击保存，在metamask中确认即可。



## 更新

主要功能已经完成，目前已经可以使用。

当前只支持ASCII字符串，后面会增加UTF-8支持，并且增加修改功能以及完善用户体验。
